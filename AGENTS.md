# AGENTS.md

## Setup
- This project uses **OpenCode** with the **Planotator** plugin.
- Configuration is stored in `opencode.json`.
- The project is a Node.js environment initialized with `npm`.

## Commands
- `npm install`: Install dependencies.
- `npm test`: Run tests (not yet configured).

## Structure
- `opencode.json`: Core agent configuration.
- `package.json`: Project manifest.
